Chillz - Movie Streaming UI
Chillz is a modern, responsive movie streaming dashboard landing page. It features a sleek dark-themed interface, movie categorization, and a featured hero section for trending content.

 Features
Responsive Sidebar: Quick navigation for Home, Favorites, Genres, and Settings.

Hero Section: Showcases featured titles (like Game of Thrones) with quick action buttons.

Search Functionality: A clean search bar to find movies instantly.

Movie Categories: Filter content by Trending, Adventure, Action, Crime, and Drama.

Movie Grid: A dynamic card layout displaying movie posters, release years, and star ratings.

Mobile Friendly: Includes a hamburger menu for smaller screens.

🛠️ Built With
HTML5: Semantic structure.

CSS3: Custom styling and layout (flexbox/grid).

FontAwesome: For UI icons.

Iconsax: For secondary vector icons.
